## 💡 调整理由

### 为什么新增 CSV/Excel 导入？

1. **用户需求强烈**
   - 用户普遍反映 JSON/YAML 格式技术门槛高
   - Excel 是最常用的办公软件
   - 数据导入是高频使用场景

2. **功能完整性**
   - 数据导入是决策分析的基础功能
   - 应该在 v1.0 之前完成并验证
   - 为 v1.0 Web UI 的数据导入做准备

3. **工作量可控**
   - CSV Loader 实现简单（0.5人日）
   - Excel Loader 工作量适中（1.5人日）
   - 可以利用现有 Loader 架构
