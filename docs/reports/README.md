# Reports Directory

This directory contains various development reports and analysis.

## Report Types

### weekly/ - Weekly Development Reports
- Weekly summaries of work completed
- Updated every Friday or at week end
- Naming: `YYYY-Www.md` (e.g., `2025-W04.md`)

### review/ - Code Review Reports
- Results from `/code-review` command
- Security and quality assessment
- Naming: `YYYY-MM-DD-{feature}.md`

### metrics/ - Metrics & Statistics
- Test coverage trends
- Performance baselines
- Dependency health analysis

## Report Templates

### Weekly Report

```markdown
# Week YYYY-Www

## Completed
- [x] Task 1
- [x] Task 2

## In Progress
- [ ] Task 3

## Next Week
- Plan item 1
- Plan item 2
```

## Auto-Generation

Some reports are auto-generated by AI commands:
- `/test-coverage` → Updates `metrics/test-coverage.md`
- `/code-review` → Creates `review/YYYY-MM-DD-{feature}.md`
- `/learn` → Updates session summaries
