# ADR Template

```markdown
# ADR-XXX: [Decision Title]

## Status
Proposed | Accepted | Deprecated | Superseded by ADR-YYY

## Context
[Describe the problem or situation that requires a decision]

## Decision
[Describe the decision made]

## Rationale
[Explain why this decision was chosen]

## Consequences

### Positive
- [Benefit 1]
- [Benefit 2]

### Negative
- [Drawback 1]
- [Drawback 2]

## Alternatives Considered

### Alternative 1: [Title]
- [Description]
- **Rejected because**: [Reason]

### Alternative 2: [Title]
- [Description]
- **Rejected because**: [Reason]

## Related
- **Issue**: [Link if applicable]
- **Commit**: [Hash if applicable]
- **Author**: hunkwk
- **Date**: YYYY-MM-DD
```

## Usage

1. Copy this template
2. Replace placeholders with actual content
3. Use sequential numbering (001, 002, 003...)
4. Save as `NNN-{short-title}.md`
5. Update `README.md` index
